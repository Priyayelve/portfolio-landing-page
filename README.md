# Portfolio Landing Page

This project is a personal portfolio landing page created using HTML and CSS. The purpose of this project is to demonstrate basic web development skills including structuring a webpage with HTML and styling it with CSS.

## Features
- Navigation bar with links to different sections
- Hero section introducing the developer
- Projects section displaying three project cards
- Footer with contact information
- Responsive layout using Flexbox

## Technologies Used
- HTML
- CSS
- Flexbox

## How to View the Project
1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Run the project using Live Server or open `index.html` in a web browser.
